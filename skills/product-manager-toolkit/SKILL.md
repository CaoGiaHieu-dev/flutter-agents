# Product Management & Business Analyst Toolkit

## When to use this skill
- Formulating a PRD (Product Requirements Document) from a problem to success metrics.
- Creating a market sizing analysis with TAM/SAM/SOM breakdown.
- Developing a cohort-based LTV (Lifetime Value) model.
- Analyzing a sales funnel or evaluating feature priority.

## How to use it (Strict Execution Rules)
1. **Hypothesis-Driven Planning:** Turn initiatives into testable hypotheses. Generate opportunity-solution trees based on Teresa Torres' method to visualize paths to user goals.
2. **Feature Prioritization:** Always evaluate features using the RICE scoring model (Reach, Impact, Confidence, Effort) before suggesting implementation.
3. **SEO & Content Strategy:** Map features/content to the buyer's journey using proven keyword modifiers:
   - *Awareness Stage:* "what is", "how to", "guide to".
   - *Consideration Stage:* "best", "top", "vs", "alternatives".
   - *Decision Stage:* "pricing", "reviews", "demo".
4. **Deliverables:** Never start coding. Output structured markdown Artifacts representing dashboards, data analyses, or PRDs.
