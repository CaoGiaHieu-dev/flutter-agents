# Python Backend & Automation Protocol

## When to use this skill
- Xây dựng API nội bộ, xử lý logic máy chủ, tự động hóa script, hoặc cào dữ liệu (web scraping).

## How to use it (Strict Execution Rules)
1. **Tiêu chuẩn Ngôn ngữ:** Bắt buộc dùng Python 3.12+ với cú pháp `async def` cho mọi tác vụ I/O.
2. **Framework:** Dùng FastAPI kết hợp Pydantic cho xác thực dữ liệu và SQLAlchemy cho ORM.
3. **Tự động hóa (Automation):** Thiết kế script theo hướng module, có argparse, log chi tiết (logger.info/error) thay vì hàm print thông thường.
